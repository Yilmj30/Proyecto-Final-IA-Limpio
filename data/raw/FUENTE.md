# Fuente del Dataset

**Nombre:** Predict Students' Dropout and Academic Success  
**Fuente:** UCI Machine Learning Repository / Kaggle  
**URL:** https://www.kaggle.com/datasets/thedevastator/higher-education-predictors-of-student-retention  
**Registros:** 4.424 estudiantes  
**Features:** 35 variables (demográficas, académicas, socioeconómicas)  
**Variable objetivo:** Target (Graduate / Dropout / Enrolled)  
**Separador:** coma (,)  
**Tamaño:** ~388 KB  
**Licencia:** CC BY 4.0  

## Descripción de variables clave

| Variable | Descripción |
|----------|-------------|
| Age at enrollment | Edad al matricularse |
| Course | Código de carrera |
| Debtor | Si tiene deuda con la institución (0/1) |
| Tuition fees up to date | Si está al día con pagos (0/1) |
| Scholarship holder | Si tiene beca (0/1) |
| Curricular units 1st sem (approved) | Materias aprobadas primer semestre |
| Curricular units 1st sem (grade) | Nota promedio primer semestre |
| Unemployment rate | Tasa de desempleo del país |
| GDP | PIB del país |
| Target | Estado: Graduate / Dropout / Enrolled |
